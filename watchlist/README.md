Watchlist
Example application for flask tutorial "Flask 入门教程".

Demo: http://watchlist.helloflask.com



Installation
$ git clone https://github.com/greyli/watchlist.git
$ cd watchlist
$ pipenv install --dev
$ pipenv shell
$ flask forge
$ flask run
* Running on http://127.0.0.1:5000/
